Definicion


El frontend o «desarrollo del lado del cliente» se refiere a la práctica de producir HTML, CSS y JavaScript. Estos tres elementos se encargan de dar forma a la parte frontal de un sitio web o aplicación. Esto incluye los fondos, colores, texto, animaciones o efectos.

Fundamentos del FrontEnd


HTML + CSS
Estos son los ingredientes básicos para construir un sitio web.


JavaScript

Un lenguaje de desarrollo front-end de programación web para crear sitios web. Gracias a JavaScript, agregas interactividad a tu sitio web, es decir, desarrollas sitios web que responden a las acciones de los usuarios.
